Cuatro reglas de diseño

Spacing

Una regla simple que se rompe constantemente. Tienes que organizar grupos de elementos similares y luego rodearlos por un espacio vacío. El espaciado puede ser una herramienta sorprendentemente poderosa.

Grid

Las dimensiones de los elementos y las distancias que los separan se organizan en una estructura cohesiva llamada grid o cuadrícula. Si un elemento queda fuera de la cuadrícula, te darás cuenta de que algo está mal.

Fonts

Para esta regla, no debe haber más de tres tipos distintos de fuente en un mismo sitio web. Una para el encabezado, otra para el cuerpo del texto y una tercera para resaltar las palabras importantes.

Colors

Además del color principal, también se usan colores de contraste para destacar detalles importantes en un sitio web. Usa las herramientas adecuadas para seleccionar una buena combinación de colores.
